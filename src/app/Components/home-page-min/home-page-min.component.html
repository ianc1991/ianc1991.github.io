MovieDB

    Front End

  • // React / JavaScript
  • // React Router
  • // Bootstrap
  • // Axios
  • // Promise Tracker

    Back End

  • // MongoDB / Mongoose
  • // Express
  • // Node
  • // JSON Web Token
  • // Bcrypt

A MERN stack project that gets data from the Mflix MongoDB database. A Node server on the back end handles database requests such as getting movie details and user authorization. JSON web tokens are used for user authorization, while Bcrypt hashes and salts passwords stored in the database. Registered users can leave comments on movie pages.
