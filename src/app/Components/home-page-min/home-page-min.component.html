  • // MongoDB / Mongoose

A current work in progress MERN stack project that gets data from the Mflix MongoDB database. A Node server on the back end handles database requests such as getting movie details and user authorization. JSON web tokens are used for user authorizaztion, while Bcrypt hashes and salts passwords stored in the database. Registered users can leave comments on movie pages.
